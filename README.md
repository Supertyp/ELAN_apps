# ELAN_apps
Apps dealing with ELAN .eaf files


## ELAN_Replacer

App that allows to find and replace specific annotations in a folder full of elan files

Dependent annotations can be used to find annotation in specific context

> :warning: **ALWAYS BACK UP YOUR DATA BEFORE USE**




## ELAN_Annotation_Splitter

App that allows to split annotations into two or more annotations together with

dependent annotations 

> :warning: **ALWAYS BACK UP YOUR DATA BEFORE USE**


## ELAN_audio_segmentation

App which takes a audio or video file as imput and creates a ELAN file with empty annotations wherever a voice is heard in the audio

> :warning: **ALWAYS BACK UP YOUR DATA BEFORE USE**
